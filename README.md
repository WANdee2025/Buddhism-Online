# Buddhism Online: Digital Contemplation Tools
### Early Buddhism in the Digital Age

This repository hosts interactive tools designed to bridge ancient wisdom with modern technology.

## 1. Bhara Sutta: The Cycle of Letting Go
A digital tool to help users understand "The Burden" (The Five Aggregates). This project uses JavaScript to visualize the process of recognizing and letting go of mental burdens.

## 2. Atta Hi Attano Natho: Be Your Own Refuge
An interactive application focused on the core principle of self-reliance. It serves as a digital reminder that inner strength is developed through one's own efforts.
