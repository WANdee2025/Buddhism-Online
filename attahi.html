อตฺตาหิ อตฺตโน นาโถ
Attā hi attano nātho, ko hi nātho paro siyā;
Attā hi sudantena, nāthaṃ labhati dullabhaṃ.

"One is truly one’s own refuge; for who else could that refuge be?"

With a self well-tamed, one gains a refuge that is hard to find.

// The Logic of Internal Sovereignty
function getRefuge(self) {
  if (self.status === 'sudantena') {
    return 'Dullabhaṃ Nāthaṃ'; // The rare refuge
  } else {
    return 'External Dependency';
  }
}
